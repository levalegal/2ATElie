# Ателье — Система учёта

Десктопное приложение для управления ателье: учёт заказов, расчёт зарплат, затраты, материалы.

**Репозиторий:** https://github.com/levalegal/2ATElie

## Стек

- Python 3.10+
- PySide6 (Qt)
- SQLAlchemy + SQLite
- openpyxl, reportlab (экспорт)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

**Логины по умолчанию:** admin / admin123, manager / admin123

## Конфигурация

Создайте `.env` (см. `.env.example`) для настройки:
- `LOW_STOCK_THRESHOLD` — порог низкого остатка материалов (по умолчанию 10)

## Тестовые данные

В разделе **Админ-панель** есть кнопка «Загрузить тестовые данные» — создаёт сотрудников, клиентов, заказы, материалы и расходы для демонстрации.

## Функции

- **Заказы:** материалы, исполнители по этапам, печать счёта
- **Зарплаты:** расчёт, история, отметка «Выплачено», учёт времени (WorkLog)
- **Материалы:** списание при использовании в заказе
- **Дашборд:** графики выручки и затрат по месяцам

## Сборка EXE

```bash
pyinstaller --clean build.spec
```

Или на Windows:
```bash
build.bat
```

Готовый файл: `dist/Atelier.exe`. База данных создаётся в папке `data` рядом с exe.

## Роли

- **Админ** — пользователи, настройки, резервные копии
- **Руководитель** — дашборд, заказы, зарплаты, затраты
- **Дизайнер** — заказы, эскизы
- **Модельер-конструктор** — заказы, лекала, материалы
- **Закройщик** — раскрой, материалы
- **Портной** — пошив, учёт времени

## Структура

- `main.py` — точка входа
- `config/` — настройки
- `database/` — модели, сессия, инициализация
- `modules/` — модули (заказы, сотрудники, материалы, зарплаты, затраты, админ)
- `ui/` — окна, стили, виджеты
- `utils/` — экспорт, валидаторы
